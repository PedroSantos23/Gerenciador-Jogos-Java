# Gerenciador-Jogos-Java
Bliblioteca de jogos - Java

Projeto em Java para gerenciamento de uma biblioteca de jogos via console.

Funcuinallidades
- Adicionar jogos
- -Listar biblioteca
- Marcar jogos como zerados
- Avaliar jogos
- Remover jogos

  Tecnologias
- Java
- ArrayList
- Scanner

  Plataforma criada para treinamento em Java
